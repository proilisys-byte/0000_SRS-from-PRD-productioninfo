#### [기존 필수 지침 유지]
- **UI 및 상태관리 필수 규칙:** 전역 상태 관리 라이브러리(Zustand)는 다크모드 설정, 네비게이션 접힘 상태 등 **'컴포넌트 트리를 3 Depth 이상 넘나드는 UI 상태'**에만 허용한다. API 데이터 통신 결과값은 절대 전역 상태 스토어에 넣지 말고, SWR/React-Query의 자체 캐싱을 이용한다.
- **성능 최적화 필수 규칙:** Prisma Schema에서 관계형 참조 필드(`xxx_id`)와 조회 쿼리 조건절에 자주 정렬/검색되는 필드(`created_at`, `status`)에만 `@@index`를 명시한다. 긴 문자열 필드(`description`, `content`)에는 인덱스를 걸지 않는다.
- **AI Fallback 필수 규칙:** AI 외부 API 호출은 최대 타임아웃 10초로 제한하며, 에러 발생 시 하드코딩된 기본 응답(Phase 3 수준)과 함께 `is_fallback: true` 필드를 반환한다. 클라이언트는 해당 플래그 수신 시 상단에 "시스템 지연으로 기본 리포트가 표시됩니다"라는 Alert 배너를 의무 노출해야 한다.
